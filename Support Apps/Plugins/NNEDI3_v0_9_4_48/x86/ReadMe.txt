Release_Intel versions are compiled with Intel Parallel Studio 2018 (icl 18.0.1.156).
Others are compiled with VS2017 Update 5.

Get x86 redistribuable package for Visual Studio 2017 Update 5.
https://go.microsoft.com/fwlink/?LinkId=746571

Get Intel Parallel Studio 2018 redistribuables to use Intel versions.
https://software.intel.com/en-us/articles/intelr-composer-redistributable-libraries-by-version

"XP" versions are compiled with Windows XP support.
"W7" versions are compiled without Windows XP support.
